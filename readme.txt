A simple plugin that creates a widget to display Facebook Page Plugin.

== Description ==

This plugin creates a widget to display Facebook Page Plugin that lets you easily embed and promote any Facebook Page on your website. Just like on Facebook, your visitors can like and share the Page without having to leave your site.

You can show/hide cover, faces and posts.

== Installation ==

For an automatic installation through WordPress:

1. Go to Plugins > Add New
2. Search for 'JGC Facebook Page Widget'
3. Click 'Install Now' and activate the plugin
4. Go to Appearance > Widgets

For a manual installation via FTP:

1. Unzip the file 'jgc-facebook-page-widget.zip'
2. Upload the 'jgc-facebook-page-widget' folder to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Appearance > Widgets

To upload the plugin through WordPress, instead of FTP:

1. Go to Plugins > Add New
2. Click 'Upload Plugin' button
3. Select 'jgc-facebook-page-widget.zip' file
4. Click 'Install Now' and activate the plugin
5. Go to Appearance > Widgets
